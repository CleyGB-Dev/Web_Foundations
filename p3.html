⚡ JavaScript: O Cérebro da Web

Aprenda a dar vida, inteligência e interatividade ao seu site.

Módulo 1: O que é JavaScript?

O JavaScript é uma linguagem de programação real. Enquanto o HTML e o CSS cuidam do visual, o JS cuida do comportamento.

Com ele, você pode criar desde botões que mudam de cor até sistemas complexos como o Facebook ou o Google Maps.

Módulo 2: Guardando Dados (Variáveis)

Imagine que variáveis são caixas onde você guarda informações para usar depois. Hoje usamos principalmente o let e o const:

let nome = "João";      // Esta caixa pode mudar de valor
const idade = 25;       // Esta caixa é fixa (constante)

O JavaScript consegue guardar textos (Strings), números (Numbers) e listas (Arrays).

Módulo 3: Funções (As Máquinas)

Uma função é um bloco de código que executa uma tarefa quando você o chama. É como uma receita de bolo: você define os passos e só "faz o bolo" quando precisa.

function dizerOla() { alert("Olá, mundo!"); }

As funções permitem que você não precise repetir o mesmo código várias vezes.

Módulo 4: Tomando Decisões (If/Else)

O JS pode decidir o que fazer dependendo da situação. É a lógica do "SE":

if (idade >= 18) {
  console.log("Pode entrar");
} else {
  console.log("Entrada proibida");
}

Módulo 5: O DOM (A Ponte)

O **DOM** é como o JavaScript "enxerga" o seu HTML. Para o JS mexer em um botão ou texto, ele precisa "selecionar" esse elemento:

  • document.getElementById("id"): Pega um elemento pelo ID.
  • document.querySelector(".classe"): Pega um elemento pela classe.

Módulo 6: Eventos (Reações)

O JavaScript fica "ouvindo" o que o usuário faz na página. Exemplos de eventos:

  • onclick: Quando clica em algo.
  • onmouseover: Quando passa o mouse por cima.
  • onkeydown: Quando aperta uma tecla.
